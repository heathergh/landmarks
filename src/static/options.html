Run on each page
⚠ warning

The Landmarks extension can show you how many landmarks there are on every page automatically, but to do so, it needs your permission to run on all pages.

About permissions

  • Permissions are needed to allow the extension tor un automatically for you on every page, rather than requiring you to activate the toolbar button or use the Landmarks extension shortcut key.

  • Granting permissions gives you:

    • Automatic counts of landmark regions on each page, displayed as a badge on the toolbar icon (if there are any landmark regions).

    • Automatic updates to the sidebar as you follow links, or switch between tabs.

  • The Landmarks extension doesn't collect any data.

  • The permissions here relate only to you allowing the Landmarks extension's code to run on all pages. If you want to see the code, you can find it in the Landmarks extension GitHub repository.

  • You can rescind these permissions at any time.

You'll need to use the toolbar button (or its shortcut key) in order to display the number of landmarks found on each page.

Note: you'll not be able to use the sidebar feature.

The number of landmarks on each page you visit will be displayed automatically in the badge of the toolbar icon.
